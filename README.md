# WeixinJSBridge

### 整理时间

2021年10月28号


### 配置环境代理说明

微信开发者工具中基本全部api都无法使用 请使用真机调试

将地址代理为pay.weixin.qq.com打开  （整理机型为ios  安卓请自行测试）
证书认证：手机浏览器输入rootca.pro （或者在调试界面点HTTPS，手机扫描即可跳转） ,会跳转到下载证书界面，点击下载好之后安装，安卓需要起个名字，ios需要在 设置——通用——关于本机——证书信任设置中开关置为开。
